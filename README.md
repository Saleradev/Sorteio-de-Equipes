// Sorteio de Equipes por Cor

Descrição do projeto:

Este projeto foi desenvolvido em **JavaScript** para atender a uma necessidade real da minha igreja: realizar um **sorteio justo e aleatório de equipes**, dividido por **duas cores**, com **limite máximo de participantes por equipe**.

O sistema garante que o sorteio seja equilibrado, controlado e simples de usar, funcionando apenas com cliques na tela.



Objetivo:

Facilitar o sorteio de equipes de forma:

 Aleatória
 Justa
 Controlada por limites
 Simples de usar em celular ou computador



Como funciona?

 O sorteio possui **duas equipes (cores)**
 Cada equipe pode ter **no máximo 50 participantes**
 A cada clique na tela:

 Uma das cores é sorteada de forma **aleatória**
 Quando uma cor atinge o limite máximo, ela **não pode mais ser sorteada**
 O sorteio continua até que **as duas equipes atinjam o limite**


Lógica utilizada

Controle de estado com variáveis
Contadores para cada equipe
Estruturas condicionais (`if / else`)
Geração de números aleatórios
Validação de limites máximos
Interação direta com o usuário por cliques



Tecnologias utilizadas

**JavaScript (puro)**
HTML (estrutura básica)
CSS
